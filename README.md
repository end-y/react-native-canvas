# react-native-canvas

React Native Canvas — imperative HTML5 Canvas 2D API powered by Skia (C++)

## Installation


```sh
npm install react-native-canvas
```


## Usage


```js
import { CanvasView } from "react-native-canvas";

// ...

<CanvasView color="tomato" />
```
